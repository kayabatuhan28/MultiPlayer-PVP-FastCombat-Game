 ------------------------------------------------------------ EN ------------------------------------------------------------
My project developed for portfolio purposes — a 1v1 PvP title played with a friend over Steam (Development time: approx. 2.5 weeks during off-work hours)

Highlighted Features of the Project (Fully designed with multiplayer focus):
- There are 2 different classes: Mage and Assassin.
- Combat is coded to be fast and fluid.
- Steam Online Subsystem was used for the multiplayer connection layer.
- Each class has 7 unique skills, and players can choose a class at the beginning of the match to start the PvP.
- I had to manually key and edit almost all animations(forward lunge, high jump, trimming frames, blending different animations, adding/adjusting keys for hand, upper arm, and clavicle bones, etc.)
- Besides skills, players have basic attacks, 4-direction dash or teleport (Assassin dash / Mage teleport), and 4-direction dodge (roll-style).
- Each skill has its own Buff or Debuff effect:
Burn: Applies damage over time by burning the target for a short duration.
Silence: Silences the target for a limited time(prevents skill usage; dash and dodge/roll can still be used).
Slow: Reduces movement speed for a short duration.
Blind: Applies a flash effect on the screen, effectively blinding the target temporarily.

LifeSteal: Steals a percentage of the damage dealt as health from the opponent.
